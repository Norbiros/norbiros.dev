# 💼 norbiros.dev

![Screenshot of norbiros.dev](public/img/readme-screenshot.png)

Personal site and blog for [Norbiros](https://norbiros.dev).

## Development

Install dependencies and start the local server:

```bash
pnpm install
pnpm dev
```

Useful checks:

```bash
pnpm lint
pnpm build
```

## Contributions

This is my personal site, so I am not accepting contributions for things like
new blog posts or changes in editorial direction. Small fixes are very welcome:
typos, broken links, accessibility issues, rendering bugs, or other focused
improvements that make the site better without changing what it is.
